## 技术栈

- web框架：blade
- 通讯框架：netty-socketio
- UI库：mdui

## 打包 

```bash
mvn clean package
```

## 运行打包程序 

```bash
java -jar target/dist/freechat/free-chat-0.0.1.jar
```
